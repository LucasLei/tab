# tab选项卡应用插件  

## tab.js
一款基于Jquery的方便易用的tab选项卡插件，可配置的事件有交换效果，鼠标点击和鼠标移上...等等！


参数  
$('').slide({

    effect: "slide",              //切换效果:fade(默认值 渐隐渐现);   
    
    liColor: "#f99",              //配置文字颜色，默认值为“#f99”;   
    
    bgColor: "green"              //是否需要配置按钮背景颜色，默认值为"green";  
    
    eventType:"mouseover"         //是否需要配置鼠标点击和鼠标移上事件，默认值为鼠标点击事件；
})
